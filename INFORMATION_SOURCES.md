# AI Tools Information Sources

## Primary Discovery Sources

These websites are verified sources for discovering new AI tools for academic research:

### 1. There's An AI For That
- **URL**: https://theresanaiforthat.com/
- **Description**: Comprehensive AI tool directory with extensive categorization
- **Update frequency**: Daily
- **Notes**: Good for finding specialized academic tools

### 2. Toolify.ai
- **URL**: https://www.toolify.ai/
- **Description**: Curated AI tools directory with user reviews
- **Update frequency**: Weekly
- **Notes**: Has academic/research category filters

### 3. AlternativeTo
- **URL**: https://alternativeto.net/
- **Description**: Community-driven software recommendations
- **Update frequency**: Continuous
- **Notes**: Good for finding alternatives to existing tools, includes user ratings

## Integrated Academic Research AI Platforms

### 4. Scite.ai
- **URL**: https://scite.ai/
- **Description**: Smart citations - shows how papers cite each other (supporting/contrasting)
- **Focus**: Citation analysis, literature validation

### 5. Scholarcy
- **URL**: https://www.scholarcy.com/
- **Description**: AI-powered summary flashcards for research papers
- **Focus**: Paper summarization, key findings extraction

### 6. Paperpal
- **URL**: https://paperpal.com/
- **Description**: AI writing assistant for academic manuscripts
- **Focus**: Grammar, language polishing, journal readiness

### 7. Writefull
- **URL**: https://writefull.com/
- **Description**: AI feedback on academic writing, integrated with Overleaf/Word
- **Focus**: Language quality, paraphrasing, title/abstract generator

### 8. Jenni AI
- **URL**: https://jenni.ai/
- **Description**: AI writing assistant with citation suggestions
- **Focus**: Academic writing, auto-completion with sources

### 9. Paperpile
- **URL**: https://paperpile.com/
- **Description**: Reference manager with AI-powered paper discovery
- **Focus**: Reference management, Google Docs integration

### 10. ResearchRabbit
- **URL**: https://www.researchrabbit.ai/
- **Description**: Visual literature discovery with citation networks
- **Focus**: Paper discovery, collaborative collections

### 11. Litmaps
- **URL**: https://www.litmaps.com/
- **Description**: Visual literature mapping and discovery
- **Focus**: Citation visualization, paper recommendations

### 12. Inciteful
- **URL**: https://inciteful.xyz/
- **Description**: Paper discovery tool using citation networks
- **Focus**: Building reading lists from seed papers
